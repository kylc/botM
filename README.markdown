botM
====

Installation
------------

From GitHub

    $ git clone git://github.com/kylc/botM
    $ cd botM
    $ cabal-dev install

From BitBucket

    $ hg clone ssh://hg@bitbucket.org/kylc/botm
    $ cd botm
    $ cabal-dev install

Example Bot
-----------

    $ dist/build/demo/demo
